# Remediation

## Spear Phishing

* Conduct regular security awareness and phishing simulation training.
* Verify the sender's identity before opening links or attachments.
* Enable Multi-Factor Authentication (MFA).
* Report suspicious emails to the security team immediately.

## Clone Website

* Always verify the website URL before entering credentials.
* Use HTTPS and check for a valid SSL/TLS certificate.
* Enable browser phishing protection and safe browsing features.
* Implement Multi-Factor Authentication (MFA) to reduce the impact of credential theft.
